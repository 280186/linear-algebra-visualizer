# MATH_CONVENTIONS.md

# 数学约定说明

本文件记录项目中所有必须统一遵守的数学约定。

线性代数可视化项目最容易出错的地方是：

1. 行向量和列向量混用；
2. 矩阵乘法顺序写反；
3. 基矩阵 P 的列含义写错；
4. 坐标变换和基变换混淆；
5. `P^{-1} A P` 与 `P A P^{-1}` 写反。

本项目统一使用以下约定。

---

## 1. 向量约定

本项目使用列向量。

向量：

```text
v = (x, y)
```

数学上视为：

```
v = [x, y]^T
```

也就是：

```
v = [ x ]
    [ y ]
```

------

## 2. 2×2 矩阵约定

矩阵 A 写作：

```
A = [ a b ]
    [ c d ]
```

其中：

```
第一列是 (a, c)
第二列是 (b, d)
```

矩阵作用于向量：

```
v' = A v
```

如果：

```
v = (x, y)
```

则：

```
x' = a*x + b*y
y' = c*x + d*y
```

------

## 3. 矩阵的列的几何意义

对于矩阵：

```
A = [ a b ]
    [ c d ]
```

标准基向量为：

```
e1 = (1, 0)
e2 = (0, 1)
```

那么：

```
A e1 = (a, c)
A e2 = (b, d)
```

因此：

```
矩阵 A 的第一列是 e1 变换后的位置。
矩阵 A 的第二列是 e2 变换后的位置。
```

这是线性变换演示模块最重要的教学点之一。

------

## 4. 矩阵乘法顺序

如果先执行变换 A，再执行变换 B：

```
v --A--> A v --B--> B(A v)
```

最终结果是：

```
v' = B A v
```

所以组合矩阵是：

```
C = B * A
```

注意：

```
先 A 后 B，对应 B * A。
```

不要写成：

```
A * B
```

------

## 5. 动画插值矩阵

第一阶段使用简单线性插值：

```
M(t) = (1 - t)I + tA
```

其中：

```
0 <= t <= 1
```

当：

```
t = 0
```

有：

```
M(0) = I
```

当：

```
t = 1
```

有：

```
M(1) = A
```

每一帧用 M(t) 作用到网格点、基向量和单位正方形。

------

## 6. 行列式

对于矩阵：

```
A = [ a b ]
    [ c d ]
```

行列式：

```
det(A) = a*d - b*c
```

几何意义：

```
det(A) 表示有向面积缩放倍数。
```

如果：

```
det(A) > 0
```

表示方向保持。

如果：

```
det(A) < 0
```

表示方向翻转。

如果：

```
det(A) = 0
```

表示二维平面被压缩到一条直线或一个点，矩阵不可逆。

程序中不要直接判断 `det === 0`，应该使用误差：

```
const EPS = 1e-8;
Math.abs(det) < EPS
```

------

## 7. 逆矩阵

如果：

```
det(A) != 0
```

矩阵 A 可逆。

对于：

```
A = [ a b ]
    [ c d ]
```

逆矩阵为：

```
A^{-1} = 1 / (ad - bc) * [  d  -b ]
                              [ -c   a ]
```

如果：

```
Math.abs(det) < EPS
```

应该认为矩阵不可逆。

------

## 8. 基矩阵

如果新基为：

```
b1 = (x1, y1)
b2 = (x2, y2)
```

则基矩阵 P 为：

```
P = [ x1 x2 ]
    [ y1 y2 ]
```

也就是说：

```
b1 是 P 的第一列。
b2 是 P 的第二列。
```

不要把 b1、b2 写成行。

------

## 9. 新基坐标到标准坐标

如果一个向量在新基 B 下的坐标为：

```
[v]_B = (u, v)
```

基矩阵为：

```
P = [b1 b2]
```

那么该向量在标准坐标下的几何位置是：

```
x = P [v]_B
```

解释：

```
[x]_E = u*b1 + v*b2
```

------

## 10. 标准坐标到新基坐标

如果已知向量在标准基下的坐标为：

```
x
```

想求它在新基 B 下的坐标：

```
[x]_B
```

公式是：

```
[x]_B = P^{-1} x
```

其中：

```
P = [b1 b2]
```

前提是：

```
P 可逆
```

也就是 b1、b2 线性无关。

------

## 11. 同一线性变换在不同基下的矩阵

假设线性变换在标准基下的矩阵是 A。

新基矩阵为：

```
P = [b1 b2]
```

那么同一线性变换在新基 B 下的矩阵是：

```
A_B = P^{-1} A P
```

解释路径：

```
[v]_B --P--> [v]_E --A--> [Av]_E --P^{-1}--> [Av]_B
```

所以整体是：

```
P^{-1} A P
```

不要写成：

```
P A P^{-1}
```

------

## 12. 常用测试矩阵

### 单位矩阵

```
[1 0]
[0 1]
```

含义：

```
不改变任何向量。
```

------

### x 方向拉伸

```
[2 0]
[0 1]
```

含义：

```
x 方向放大 2 倍。
```

------

### y 方向拉伸

```
[1 0]
[0 2]
```

含义：

```
y 方向放大 2 倍。
```

------

### 逆时针旋转 90 度

```
[0 -1]
[1  0]
```

含义：

```
e1 -> (0, 1)
e2 -> (-1, 0)
```

------

### 水平剪切

```
[1 1]
[0 1]
```

含义：

```
e1 -> (1, 0)
e2 -> (1, 1)
```

------

### 压缩到 x 轴

```
[1 0]
[0 0]
```

含义：

```
所有点的 y 坐标变成 0。
矩阵不可逆。
```

------

### 关于 y 轴翻转

```
[-1 0]
[ 0 1]
```

含义：

```
x 坐标取反，y 坐标不变。
det(A) = -1，方向翻转。
```

## 13. 特征值与特征向量

特征值 λ 和特征向量 v 满足：

```
A v = λ v
```

其中 v 是非零向量。

特征向量的几何意义：

```
经过矩阵 A 变换后，特征向量 v 仍然保持在同一条方向直线上。
λ > 0：方向保持
λ < 0：方向反向
|λ| > 1：长度放大
0 < |λ| < 1：长度缩小
|λ| ≈ 0：压缩到零向量
```

特征值计算（2×2 矩阵）：

```
A = [a b]
    [c d]

trace = a + d
det = ad - bc
discriminant = trace² - 4·det

discriminant > EPS：两个不同实特征值
  λ₁ = (trace + √discriminant) / 2
  λ₂ = (trace - √discriminant) / 2

|discriminant| ≤ EPS：重特征值
  λ = trace / 2
  可能是所有方向都是特征方向（A ≈ λI）
  或只有一条特征方向

discriminant < -EPS：无实特征值
  （例如旋转矩阵）
```

特征向量求法：

```
对每个特征值 λ，求 (A - λI) 的零空间
M = A - λI
如果行向量为 [p, q]，零空间方向为 (-q, p)
归一化后即为特征方向
```

------

## 14. 编码时的重点提醒

1. 永远确认当前使用的是列向量约定；
2. `multiplyVec` 应该实现 `A v`；
3. `multiplyMat` 的语义必须清楚；
4. 先 A 后 B，一定是 B * A；
5. 基矩阵 P 的列是基向量；
6. 坐标变换使用 `P^{-1} x`；
7. 同一变换换基使用 `P^{-1} A P`；
8. 所有接近 0 的判断都使用 EPS。
