## 七、关键数学约定

请严格遵守：

1. 列向量约定
2. 先 A 后 B 对应 B*A
3. 基矩阵 P = [b1 b2]，b1 是第一列
4. A_B = P^{-1} A P，不是 P A P^{-1}
5. EPS = 1e-8 判断不可逆
6. 不显示 NaN / Infinity / -0.00
