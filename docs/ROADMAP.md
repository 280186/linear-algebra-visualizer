## Stage 4.7.2 详细说明

### 目标

在 Canvas 中直观演示：

```
[v]_B --P--> v_E --A--> A v_E --P^{-1}--> [Av]_B
```

### 实现建议

1. 仍然使用同一个 Canvas
2. 不做多画布
3. 不引入 SVG、KaTeX、WebGL、PixiJS
4. 不做特征值
5. 先只对选中的一个用户向量做路径动画
6. 控制面板负责步骤说明
7. Canvas 负责轻量高亮和向量移动

### 路径步骤

```
step 0: 显示 [v]_B
step 1: P [v]_B = v_E
step 2: A v_E
step 3: P^{-1} A v_E = [Av]_B
step 4: 对比 A_B [v]_B
```

## Stage 4.8 详细说明

实现特征值和特征向量的可视化。

### 核心概念

- 特征值 λ：Av = λv
- 特征向量 v：变换后仍在同一直线上
- 特征多项式：det(A - λI) = 0
